<HTML + JAVASCRIPT : MANIPULAÇÃO> 
<ENTRADA DE DADOS [INPUT]>

Utilize esse metodo para coletar um valor numérico contido dentro
de um objeto HTML e atribui-lo a uma variavel JAVASCRIPT ::

var sec = Number(window.document.getElementById('txtime').value)

var :: cria espaço na memória

sec :: nome do objeto

Number() :: converte todo o conteudo para um tipo numérico

window.document.getElementById('txtime')

.value :: coleta o valor contido dentro da entrada (input)
<CONTEUDO INTERNO>

Utilize esse metodo caso queira mudar o valor interno de um
objeto HTML utilizando JAVASCRIPT ::

var res = window.document.querySelector('p#res')
Coletando objeto e colocando dentro na variavel res

res.innerHTML = sec + "s" ou res.innerHTML = 15
Utilize o sistema de escrita de texto do JAVASCRIPT para renomear.
