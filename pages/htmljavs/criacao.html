<CRIANDO UM ELEMENTO IMAGEM [IMG]>

var space = document.querySelector('div#space') ::
coloca a div#space já criada no HTML dentro da variavel
space, para que possamos utiliza-la em outra parte do projeto.

var img = document.createElement('img') :: Equivalente 
a criar tag img no documento HTML

img.setAttribute('id','foto') :: Atribui ao elemento img
criado uma id chamada foto.

img.setAttribute('src','imagens/jovem-h.png') :: adiciona
o endereço da imagem pelo atributo src no objeto img.

space.appendChild(img) :: pegamos a variavel space que agora é
nossa div#space, e atribuimos a ela o filho(img) para que apareça
em tela.
