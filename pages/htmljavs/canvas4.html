<CURVA BEZIER>
:: Exemplo em um canvas 400x100

let inicial = { x: 50, y: 20 }; // moveTo
let pc = { x: 230, y: 30 }; // A
let sc = { x: 150, y: 80 }; // B
let final = { x: 250, y: 100 }; // final

ctx.beginPath();
ctx.moveTo(inicial.x, inicial.y);
ctx.bezierCurveTo(pc.x, pc.y, sc.x, sc.y, final.x, final.y);
ctx.stroke();

Os argumentos bezierCurveTo(...) são:

1º (A) Coordenada de controle X
2º (A) Coordenada de controle Y

3º (B) Coordenada de controle X
4º (B) Coordenada de controle Y

5º Coordenada final do eixo X
6º Coordenada final do eixo Y
