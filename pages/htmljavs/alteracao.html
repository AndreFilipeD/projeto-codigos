<ALTERAR IMAGEM HTML VIA JAVASCRIPT>

Considere a seguinte tag em HTML ::
<img id="image" src="teste.jpg">

Importe esse img pelo id "#image" para uma variavel ::
var img = document.querySelector("#image");

Defina no atributo Source 'src' o endereço, ex ::
img.setAttribute('src', 'outra_imagem.png');
