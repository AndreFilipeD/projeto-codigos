<JS : ORIENTAÇÃO A OBJETOS>
<CLASSE E METODO CONSTRUTOR>

    class Pessoa {
        constructor(nome, idade){
            this.name = nome
            this.age = idade
        }
    }

:: Acabamos de criar uma nova classe de objetos, para que
assim quando necessário, podemos instancia-lo(cria-lo) quantas 
vezes quiser e definir suas propriedades durante a criação 
utilizando o metodo constructor dentro da classe.

o metodo constructor receberá em sua criação como parâmetro
o nome e a idade, e atribuirá nas variaveis respectivamente
name e age, conforme definido no metodo constructor.
<CRIANDO OBJETOS>

let p1 = new Pessoa("andré", 28)
let p2 = new Pessoa("maria", 38)
let p3 = new Pessoa("joao", 18)

:: com a classe e o metodo construtor já definido acima, apenas
isso já é o suficiente para criar 3 objetos com name e age
atribuidos.

window.alert(p1.name) :: resultado "andré"
window.alert(p2.age) :: resultado 38
