<JS : VARIAVEIS> 
<TIPOS>

var :: pode ser declarada em qualquer situação e ser utilizada
antes mesmo de ser declarada, e funciona dentro e fora de seu escopo
sem quaisquer problemas.

let :: é baseada em escopo de blocos, fazendo com que funcione
apenas dentro de seu escopo declarado.

const :: deve ser inicializada imediatamente após sua declaração, pois
é uma variavel constante, e não poderá ser alterada, também tem escopo
de bloco igualmente a variavel let.


<DECLARAÇÃO>

var nome = "andre" :: typeof nome :: string
var salario = 0 :: typeof salario :: number
var sorte = false ou f :: typeof sorte :: boolean

let idade = 25 :: escopo de bloco

const gravidade = 9.807 :: constante, escopo de bloco

<CONVERSÃO BASICA>

Number.parseInt(variavel) :: converte a variavel para inteiro.
Number.parseFloat(variavel) :: converte a variavel para float.

Number(variavel) :: JS se vira pra converter em numero.

variavel.toString() :: converte variavel numerica para string.

variavel.toLocateString('pt-BR',{style:'currency',currency:'BRL'}) ::
Utiliza o padrão monetario Brasileiro na apresentação do valor;
Use em 'BRL' o 'USD' para dolar, 'EUR' para euro entre outros...

<CONVERSÃO DECIMAL/HEXADECIMAL>

Convertendo valor decimal inteiro para hexadecimal String ::
hexString = yourNumber.toString(16);

Convertendo valor hexadecimal string para decimal Inteiro ::
yourNumber = parseInt(hexString, 16);
