<JS : FUNÇÕES>
<STANDARD>

function somar(x, y){
    return x + y;
}
:: metodo tradicional de criação de funções em javascript

<ARROW FUNCTION>

const somar = (x, y) => {
    return x + y;
}
:: É declarada igual uma variavel, mas é uma função


const somar = (x, y) => x + y;
:: Pode resumir, caso declare em uma só linha, e não escreva 
return e nem coloque as chaves {} sobre o conteúdo.


const duplicar = x => x * 2;
:: Pode resumir ainda mais, caso seja só um parâmetro de entrada.


essa mesma função acima de duplicar em standard 
seria escrita assim ::

function duplicar(x){
    return x * 2;
}

<FUNÇÃO DENTRO DE UM OBJETO>


let pessoa = {
    nome: 'andre',
    sobrenome: 'domingues',
    idade: 28,
    nomeCompleto: function(){
        return this.nome + ' ' + this.sobrenome
    }
}


:: Utilizando a propriedade this, pode acessar os valores
do próprio objeto, e assim criar uma função para retornar
valores.

Para retorna-la execute como o exemplo abaixo ::

window.alert(pessoa.nomeCompleto())

:: Lembrando que se utilizar uma arrow function dentro
de um objeto, não terá acesso ao this., e
por isso não poderá acessas as propriedades do objeto.
