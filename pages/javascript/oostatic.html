<JS : ORIENTAÇÃO A OBJETOS STATIC>
<ENTENDENDO STATIC>

class Person{
    static hands = 2
    constructor(nome){
        this.name = nome
    }
    
    static sayHi(){
        window.alert(`Olá, ${this.name}`)
    }
}


:: A declaração static sinaliza que a classe
Person terá determinado valor, então, para
verificarmos quantas mãos qualquer Person tem devemos fazer ::

Person.hands ao invez de this.hands
:: Como se trata de um valor static, está referenciando a classe
do objeto, e não o objeto em si, por isso this não funciona.

Também podemos alterar os valores static, mas ao invez de ::

this.hands = 3;

utilizamos ::

Person.hands = 3;
E ao tentar utilizar o metodo sayHi();
retorno :: 'Olá Person'

:: Como se trata de uma função estática, ao coletar o this.name
, pegamos o nome da classe, já que se refere a classe toda e
não a um objeto gerado a partir da classe.
