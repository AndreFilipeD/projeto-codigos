<GETTER>

Considere a seguinte classe ::

    class Person{
        age = 10
        steps = 0
        constructor(primeiroNome, segundoNome){
            this.firstName = primeiroNome
            this.secondName = segundoNome
        }
        takeAstep(){
            this.steps++
        }
    }

:: Para utilizar getter (e setter) geralmente utilizamos
o nome da própria variavel na função, e ai comunalmente 
adcionamos um "_" no inicio do nome da variavel para
não haver conflito, conforme exemplo abaixo ::

_age = 10;
agora vamos criar dentro da class, o metodo get ::

get age(){
    return this._age;
}
p1 = new Person("João", "Pedro");

window.alert(p1.age); :: resultado será 10, utilizando get, podemos
processar a variavel inicialmente dentro da classe do objeto
e depois retornar algum valor já otimizado para determinadas
situações.
Agora vamos criar outro get dentro de class para gerar um nome completo ::

get fullName(){
    return `${this.firstName} ${this.secondName}`;
}

window.alert(p1.fullName) :: retonará "João Pedro"
