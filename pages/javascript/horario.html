<JS : HORÁRIO> 
<INICIALIZAÇÃO DATE>

var d = new Date() :: receberá data do navegador
e atribuirá a uma variavel
<DATE PERSONALIZADA>

new Date(year,month,day,hours,minutes,seconds,ms)
example :: new Date(2018, 11, 24, 10, 33, 30, 0);

Remova os ultimos parâmetros na ordem caso não queira especificar
todas as informações como nos exemplos abaixo ::

new Date(year,month,day,hours,minutes)
ou
new Date(year,month)
<METODOS DATE>

getFullYear() :: ano (yyyy)
getMonth() :: mês (0-11)
getDate() :: dia do mês (1-31)
getDay() :: dia da semana (0-6)
getHours() :: hora (0-23)
getMinutes() :: minuto (0-59)
getSeconds() :: segundo (0-59)
getMilliseconds() :: milisegundos (0-999)

getTime() :: Retorna um número representando
os milissegundos passados entre 1 de Janeiro de 1970 00:00:00
UTC e a data atual.
<UTILIZANDO METODOS DATE>

var d = new Date() :: criando date

var hora = d.getHours() :: atribuindo horario de date ['d']
na variavel hora.
