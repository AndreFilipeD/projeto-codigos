<JS : EVENT LISTENER> 
<ESTRUTURA>

target.addEventListener("event", function, useCapture)

target :: é o elemento HTML a ser manipulado.

.addEventListener(...) :: Elemento que faz parte do DOM
que serve como manipulador de eventos.

event :: string que especifica qual evento deseja ouvir
Caso queira ver quais são os eventos, clique aqui

function :: função sem parenteses a ser executada

useCapture :: um valor booleano opcional por padrão false
que especifica se o evento deve ser executado na fase de 
captura ou de envio aos eventos pais
<EXEMPLO>

var botao = window.document.getElementById('botao');
var size = 75;

botao.addEventListener("click", crescer);

function crescer(){
    size+=10;
    botao.style='width:'+(size/1.5)+'px;
    height:'+size+'px;
    border-radius:'+(size*2)+'px;'
}
