Para exemplificar o conceito de herança, vamos criar uma
classe para pessoas ::

class Pessoa{
    age = 0
    constructor(nome){
        this name = nome
    }
}
:: Todo aluno é uma pessoa, correto? então se criar-mos uma
classe para Alunos, podemos extender o que já foi criado na classe
pessoas, correto?

class Aluno extends Pessoa{
    escola = 'senai'
    constructor(nome, id){
        super(nome)
        this.id = id
    }
}

:: Como pode observar, o aluno já tem um nome que foi herdado
da classe pai Pessoa, e para acessa-lo utilizamos super() no
construtor da classe Aluno. também acrescentamos o campo escola
e o ID do aluno. no caso de age, não há necessidade de utilizar
o super(), pois ele foi declarado fora do construtor de Pessoa.

super sempre acessará propriedades do Pai, podendo
pegar valores e até funções declaradas pelo mesmo,
para que possa ser utilizada pelo filho.
