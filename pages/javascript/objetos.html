<JS : OBJETOS> 
<INICIALIZAÇÃO>
Objetos podem ser declarados com var, let e const

var roupa = {
    nome:"Camisa social", //string
    cores:["branco","preto"] //array
    peso:"300g", //string
    tamanho:40, //number
    propriedades: function(){
        return this.cor + ' ' + this.peso + ' ' + this.tamanho
    } //retorna: branco 300g 44
    estampa:{ // objeto dentro do objeto
        altura:10, dentro do objeto
        largura:15, //number dentro do objeto
        desenho:"dragão" //string dentro do objeto
    }
}

dentro de um objeto, podemos armazenar
Numeros, Strings, Arrays, funções e até 
outros objetos contendo tudo isso

<ACESSANDO CONTEUDO> 

Agora vamos acessar os dados declarados no objeto acima ::

window.alert(roupa.nome) ::::::::::::: "camisa social"
window.alert(roupa.cores[0]) ::::::::: "branco"
window.alert(roupa.peso) ::::::::::::: "300g"
window.alert(roupa.tamanho) :::::::::: 40
window.alert(roupa.propriedades()) ::: "branco 300g 44"
window.alert(roupa.estampa.desenho) :: "dragão"
<CLONANDO OBJETOS> 
Considere os seguintes arrays de objetos ::

frutas=[ //array com 2 objetos
{nome:"banana",quantidade:3,peso:"70g"},
{nome:"abacaxi",quantidade:1,peso:"200g"}
]

estoqueFruta=[]

Utilizando :: (Object.assign{}, /*objeto a ser clonado*/ )

Vamos clonar o abacaxi para o estoqueFruta ::

estoqueFruta.push(Object.assign({},frutas[1])); 
//Inclui objeto abacaxi na ultima posição da lista.

estoqueFruta.unshift(Object.assign({},frutas[1]));
//Inclui objeto abacaxi na primeira posição da lista.
Também pode inserir novas propriedades no clone do objeto
utilizando dentro dos colchetes ::

(Object.assign({altura:"20cm"},frutas[1]))
