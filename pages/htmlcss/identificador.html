<HTML + CSS : IDENTIFICADORES>
<CLASSE E ID>

<h1 id="titulo"></h1> :: Id só pode ser utilizada 1 vez.
<p class="parag"></p> :: Classe não há limites de uso.

Utilize os identificadores conforme o exemplo abaixo:

h1#titulo{
    color:...
    background:...
}

p.parag{
    color:...
    background:...
}
[.] representa CLASSE
[#] representa ID
[:] representa PSEUDO-CLASSES
[::] representa PSEUDO-ELEMENTOS
[::] representa CHILDREN'
<MULTICLASSE>

<h1 class="primeira segunda"> Essa tag tem duas classes <h1>

Para utilizar duas classes em uma unica tag
acrescente espaço entre elas como no exemplo,
e utilize em css sem problemas ::

.primeira{
    font-size: 2em;
}
.segunda{
    font-weight: 600;
}
