<HTML5 : IFRAME> 
<PERMITE CRIAR UMA CAIXA COM UMA PAGINA WEB DENTRO>

<iframe src="pagina01.html" frameborder="0" height="500" width="500">
   Link alternativo ou mensagem de incompatibilidade;
</iframe>

Online :: se não houver bloqueio de segurança
Offline :: de algum diretório pessoal
<RODANDO ESSE MESMO SITE DENTRO DE UM IFRAME> ::

<iframe src="https://andrefiliped.github.io/projeto-codigos/" frameborder="0" height="100" width="500">
    Link alternativo ou mensagem de incompatibilidade;
</iframe>


<ESCREVENDO DENTRO DO IFRAME SEM CRIAR UMA NOVA PÁGINA> ::

<iframe srcdoc="<h1>Titulo</h1><p>Estou escrevendo dentro do iframe!!</p>"
frameborder="1" height="100" width="300">
   Link alternativo ou mensagem de incompatibilidade;
</iframe>


<UTILIZE EM SEU IFRAME PARA CONFIGURAR SUA SEGURANÇA> ::

<iframe src="pag01.html" referrerpolicy="no-referrer" sandbox="sandbox">
    Link alternativo ou mensagem de incompatibilidade;
</iframe>

sandbox="allow-scripts" :: permite o uso de scripts no iframe;

sandbox="allow-same-origin" :: Sem limites contanto que no diretório origem;

sandbox="allow-forms" :: Permite a manipulação de formulários.

<EM CSS>
overflow: none;
ou
overflow-y: none;
overflow-x: none;
:: deve esconder barras de rolagem do iframe
