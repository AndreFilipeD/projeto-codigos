<HTML5 : INICIALIZAÇÃO> 

<DOCTYPE html> :: declarar criação de um arquivo HTML5
essa parte é a primeira linha do HTML;

<html lang="pt-br">...</html> :: abaixo do primeiro comando, 
envelopa todo o código html e define sua linguagem;

<head>...</head> e <body>...</body> :: são os dois blocos 
principais, que estão contidos pelo bloco html acima.. onde head 
é o controle, e body o conteudo;

<meta charset="UTF-8"> :: dentro de head, 
comunica ao navegador o formato de codificação de caracteres

<meta name="viewport" content="width=device-width, initial-scale=1.0"> :: 
detro de head, define o formato de exibição e escala da pagina

<title>Meu Site</title> :: detro de head, nome da pagina no navegador

<link rel="shorcut icon" href="favicon.ico type="image/x-icon"> ::
detro de head, importa miniatura icone.ico no navegador

<Exemplo de inicialização>

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Meu site</title>
</head>
<body>
    
</body>
</html>
