<HTML5 : TAGS FORMULÁRIOS> 
<form action="cadastro.js" autocomplete="on" method="get">...</form> ::
envelopa o formulário ; Action roda o script após o envio do formulário.

method="get" ou "post", no qual get mostrará os dados do formulário na URL,
ajudando caso queira compartilhar link, e post ocultará esses dados, que
ainda podem ser encontrados.. se quiser segurança precisa aprender HTTPS.

autocomplete="on" ou "off" retira ou coloca aquela caixa de
sugestão de informações previamente já escritas.
<label for="inome">Nome:</label> :: Rótulo do input, vinculado pelo for e id
do input, no caso label for="inome" e input id="inome";
<input type="text" name="nome" id="inome"> :: Clicar no label seleciona input.
<input type="text" required> :: só libera envio formulário após preenchido.
<input type="text" minlenght="5"> :: Só libera tendo o minimo exigido.
<input type="text" maxlenght="15"> ::  Só libera até o maximo exigido.
<input type="text" size="10"> :: Não trava, apenas ajusta tamanho da celula.
<input type="text" placeholder="Digite Aqui"> :: Palavra instrucional na caixa.
<input type="text" value="André"> :: Valor inicial ou nome de botão.
