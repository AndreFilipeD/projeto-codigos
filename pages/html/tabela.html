<HTML5 : TAGS TABELAS> 
<TODAS AS TAGS>

<table>...</table> :: Cria a tabela,
todo o conteudo de tabela deve estar dentro dessa tag.

<thead>...</><tbody>...</><tfoot>...</> :: Como o header,
main e o footer da pagina web, elas servem para organizar o
conteudo da tabela de maneira semantica e organizada.

<tr>...<tr/> :: Define uma linha na
tabela para todo conteudo que estiver dentro.

<td>...<td/> :: Define o conteudo dentro da celula
na tabela, que deverá estar dentro de uma linha <tr>.

<th>...<th/> :: Define celula de titulos semanticamente
dentro da tabela, e deverá estar dentro de uma linha <tr>.

<ROWSPAN E COLSPAN>

<td rowspan="2"> 75 </td> :: Faz com que essa celula ocupe 
+1 linha abaixo, se fosse colocado em rowspan="3", ele ocuparia
+2 linhas abaixo, e assim sucessivamente...

<td colspan="2"> 35 </td> :: Faz com que essa celula ocupe 
+1 coluna a direita, se fosse colocado em rowspan="3", ele ocuparia
+2 colunas a direita, e assim sucessivamente...

<th rowspan="2"> 75 </th> e <th colspan="2"> 75 </th> :: faz a mesma
coisa das de cima, só que semanticamente <th> é para títulos.

<SCOPE ROW E COL>

<th colspan="4" scope="row"></th> :: o comando scope="row" diz
que esse titulo <th> está semanticamente apresentando o conteudo
da linha.

<th rowspan="3" scope="col"></th> :: o comando scope="col" diz
que esse titulo <th> está semanticamente apresentando o conteudo
das colunas á direita.

<SCOPE ROWGROUP E COLGROUP>

<th rowspan="2" scope="rowgroup"></th> :: o comando scope="rowgroup"
é igual o "row" e o "col", mas não apresenta conteudo, e sim um 
conjunto de titulos abaixo da celula.

<th rowspan="2" scope="colgroup"></th> :: o comando scope="colgroup"
é igual o "row" e o "col", mas não apresenta conteudo, e sim um 
conjunto de titulos á direita da celula.
