<HTML5 : LISTAS>
<LISTA ORDENADA>

<ol>...</ol> :: Listas ordenadas por caracteres
<ol type="1"> :: Sequencia numérica 1,2,3...
<ol type="a"> :: Sequencia minuscula a,b,c...
<ol type="A"> :: Sequencia MAIUSCULA A,B,C...
<ol type="x"> :: Romano minuscula iii,iv,v...
<ol type="X"> :: Romano MAIUSCULA III,IV,V...

<ul>...</ul> :: Listas ordenadas por pontos
<ul type="circle">...</ul> :: Ponto redondo
<ul type="disk">...</ul> :: Ponto disco
<ul type="square">...</ul> :: Ponto quadrado
<ul type="none">...</ul> :: Sem ponto

<li>item</li> :: Tanto em ol quando em ul,
utilize essa tag dentro para definir os itens que estarão 
contidos dentro da lista - tag final com barra é dispensavel.

<LISTA DE DEFINIÇÃO>

<dl>...</dl> :: Lista de definição
<dt>...</dt> :: Termos
<dd>...</dd> :: Definições
Dentro de dl, voce coloca os termos, e suas definições abaixo,
igual no exemplo a seguir ::
<dl>
    <dt>Veiculos</dt>
    <dd>Moto</dd>
    <dd>Carro</dd>
    <dt>Marca</dt>
    <dd>Honda</dd>
</dl>

Essa lista ficará assim ::

Veiculos
Moto
Carro
Marca
Honda
<LISTA DADOS DE PESQUISA>

<input list="browsers" name="browser"/>
<datalist id="browsers" placeholder="clique aqui">
    <option value="Chrome">
    <option value="Microsoft Edge">
</datalist>

Essa lista ficará assim ::



    Reage conforme a pesquisa, gerando uma lista.
