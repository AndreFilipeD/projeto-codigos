<picture> :: tag para trabalhar com imagens responsivas,
ela deve envelopar o conteudo abaixo.

<source media="(max-width: 750px)" srcset="imagens/peq.png" type="image/png"> ::
Se o tamanho for no maximo 750px, importe deste endereço 

<source media="(max-width: 1050px)" srcset="imagens/med.png" type="image/png"> ::
Caso contrario, se o maximo for 1050px, importe esse endereço

<img src="imagens/gra.png" alt="imagem flexivel"> ::
Se nenhuma das 2 acima aparecer, esse img normal mostrará essa,
que é maior que as outras, gerando um dinamismo na página.
