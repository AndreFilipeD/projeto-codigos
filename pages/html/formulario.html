<RADIO>

Para que todos os radios estejam conectados, utilize
em todos eles o name="igual", pois se for "diferente"
seria como se fosse outro formulário separado;

Caso queira um titulo clicavel, adicione uma id no radio
e depois crie um label com o for apontado pra esse id criado;

Confira o exemplo abaixo ::

<form autocomplete="on">
<input type="radio" name="web" value="sim" id="idsim"><label for="idsim">Sim</label>
<input type="radio" name="web" value="nao" id="idnao"><label for="idnao">Não</label>
<input type="radio" name="web" value="talvez" id="idtalv"><label for="idtalv">Talvez</label>
<input type="radio" name="web" value="quem_sabe" id="idqs"><label for="idqs">Quem Sabe</label>
</form>









Atribua em radio checked="checked" para definir um radio
selecionado inicialmente por padrão, conforme exemplo abaixo ::

<input type="radio" name="imgsel" value="" checked="checked" />
