<HTML5 : SELECT>
<INICIALIZAÇÃO>

A tag <select> também é uma entrada de formulario, então
deve ser envelopada por um <form>...</form>.

<select name="jogos">
    <option value="">Escolha uma plataforma de jogos</option>
    <option value="xbox">Xbox Series</option>
    <option value="ps5">PS5</option>
    <option value="pc">PC Gaming</option>
</select>



<select name="plataforma" required="required">...</select> :: acrescente
required para garantir que alguma opção foi selecionada antes do envio.

<OUTRAS CONFIGURAÇÕES>

<select name="plataforma" multiple>...</select> ::
Ao segurar "ctrl", pode selecionar mais de uma opção, ex ::



<select name="plataforma" size="5">...</select> ::
Utilize size para definir o tamanho da lista, assim evitando ter que scrollar.



Seletor de sim ou não simples ::
<label>Você tem mais de 18 anos?</label>
<select name="maioridade">
    <option value="sim">Sim</option>
    <option value="nao">Não</option>
</select>
