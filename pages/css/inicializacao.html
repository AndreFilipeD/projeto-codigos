<CSS3 : INICIALIZAÇÃO>
<INICIALIZAÇÃO INTERNA> [.class && #id]

<style>...</style> :: 
Insira esse bloco dentro de <head>...</head>, assim poderá
criar o estilo diretamente dentro dessa tag, sem a
necessidade de um arquivo externo.
<INICIALIZAÇÃO EXTERNA(RECOMENDADO)>

<link rel="stylesheet" href="estilos/style.css"> ::
Após inserir essa tag dentro de <head>...</head>,
insira dentro do arquivo, no diretório selecionado em href,
a seguinte instrução ::

@charset "UTF-8"; ::
Define o formato de codificação de caracteres.
<INICIALIZAÇÃO INLINE>

Exemplos de inicialização inline ::

<h1 style="color:orange;padding:30px;"> Titulo </h1>

E ::

<p style="color:purple;font-weight: 600;"> Parágrafo </p>
<DICA>

    *{
        margin: 0px;
        padding: 0px;
        box-sizing: border-box;
    }
    
Utilize esse comando dentro de *, que acessará todas as caixas e
definirá inicialmente todas elas sem espaçamentos para que você
possa customizar de maneira mais precisa;

box-sizing: border-box; :: tudo que for aplicado no objeto filho
não irá transpassar o objeto pai. É bom inicializar como padrão.
