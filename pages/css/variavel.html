<CSS3 : VARIAVEL>
<DECLARAÇÃO>

Podemos armazenar praticamente qualquer propriedade
dentro de uma variavel;

Defina as variaveis em CSS dentro de :root ::

:root{
    --fonte1: Verdana, Geneva, Tahoma, sans-serif;
    --cor1: #121314;
    --main-border-radius: 20px
    --fontsize: 2.34em;
    --image-width: 800px;
    --image-height: calc(var(--image-width) / (4/3));
    --numerocalc: 10;
    --all-padding: 60px 20px 40px 10px;
    --margin: auto;
    --main-image: url(local/file.png);
}
<APLICAÇÃO>

#blockexample{
    font-family: var(--fonte1);
    color: var(--fonte1);
    border-radius: var(--fonte1);
    font-size: var(--fontsize);
    width: var(--image-width);
    height: var(--image-height);
    padding: var(--all-padding);
    margin: var(--margin);
    background-image: var(--main-image);
}
