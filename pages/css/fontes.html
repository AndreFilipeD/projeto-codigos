<CSS3 : FONTES>
<IMPORTAÇÃO> :: vem a seguir de charset ::

@import url('https://fonts.googleapis.com/
css2?family=Dancing+Script&display=swap') ::
Importa do google fonts. 

font-family: 'Dancing Script','cursive'; ::
adicione a familia á cascata que deseja usar a fonte importada


    @font-face {
        font-family: [none];
        src: url('fontes/lovelarry.otf');
        format('opentype');
    }
    
    @font-face {
        font-family: [none];
        src: url('fontes/lovelarry.ttf');
        format('truetype');
    }

    talvez seja necessário colocar o font-weight e
    font-style como normal para padroniza-las.
<TIPOS DE FORMATO> ::

opentype (otf) - truetype (ttf) - embedded-opentype 
truetype-aat (Apple Advanced Typ) - (svg)


<TEXTO>

text-decoration: none ou underline; :: decoração do texto.

text-indent: 30px; :: margem inicial a esquerda do texto

text-allign: justify; :: distribui as palavras
igualmente dentro da caixa de texto, a fim de preenche-la.

<FONTE>

font-family: Arial, Helvetica, sans-serif; ::
sempre utilizar fontes fundamentais, caso não consiga
carregar a primeira fonte principal, safe combinations.

font-weight: lighter ou normal ou bold ou bolder
ou 100 a 900; :: grossura, intensidade da fonte.

font-style: normal ou italic; :: estilo da fonte.

<EXTRA>

margin: auto; :: centraliza texto ao centro.
