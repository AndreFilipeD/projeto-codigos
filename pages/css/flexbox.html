<CSS3 : FLEXBOX>
Para organizar o conteúdo, devemos "encapsular" em uma box
e atribuí-la :: display: flex; :: e o conteudo se alinhará.

flex-direction: row; :: faz o conteudo interno da caixa
se alinhar em linha(horizontal).

flex-direction: column; :: faz o conteudo da caixa se
apresente em coluna(vertical).

Pode ordenar o conteudo das linhas e colunas usando o seguinte ::
style="order: 1;" style="order: 0;"style="order: 2;"...

flex-wrap: wrap; :: quebra de elementros dentro do flex se
houver dimensionamento de tela sobre eles.
<Manipulando conteudo dentro do espaço da flex>

justify-content: center; :: conteudo centralizado.
justify-content: flex-start; :: conteudo alinhado no começo.
justify-content: flex-end; :: conteudo alinhado no final.
justify-content: space-between; :: distribuido sem laterais.
justify-content: space-around; :: cada objeto tem espaço fixo.
justify-content: space-evenly; :: distribuido igualmente.

allign-itens: stretch; :: estica pelo espaço disponivel.
allign-itens: center; :: centraliza em seu espaço.
allign-itens: start; :: aparece á esquerda do espaço.
allign-itens: end; :: aparece á direita do espaço.
