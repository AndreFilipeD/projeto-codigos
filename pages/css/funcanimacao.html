<CSS3 : FUNÇÕES DE ANIMAÇÃO>
<KEYFRAMES>

@keyframes nomeanimacao{} :: Criamos um animation name para ser
importado em alguma caixa.

@keyframes example{
    from {background-color: red;}
    to {background-color: yellow;}
} :: definimos como começa e termina a animação.

@keyframes example{
    0% {background-color: red;}
    50% {background-color: green;}
    100% {background-color: blue;}
} :: podemos selecionar por % da animação.
<FUNÇÕES>

animation-name: example; :: Importa a animação
pelo nome definido no keyframe da mesma.

animation-duration: 4s; :: Define a duração da
animação.

animation-delay: 1s; ou -2s; :: Define um atraso
ou se negativo um adiantamento na inicialização da animação.

animation-iteration-count: 3; ou infinite; :: Define quantas
vezes a animação será repetida.

animation-direction: normal; ou reverse; ou alternate; ou
alternate-reverse; :: Modifica a direção de inicio da animação,
e pode ser configurado alternar, ou alternar do final.

animation-timing-function: ease; ou linear; ou ease-in;
ou ease-out; ou ease-in-out; ou cubic-bezier(n,n,n,n); ::
Ajusta a velocidade de como a animação vai correr em cada etapa.

animation-fill-mode: none; ou forwards; ou backwards; ou both; :: 
Define se a animação vai inicializar ou finalizar
"vestindo" o estilo aplicado no keyframe.
