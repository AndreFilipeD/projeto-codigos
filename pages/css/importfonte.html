<CSS3 : IMPORTAR FONTES>
<IMPORTAÇÃO> :: vem a seguir de charset ::

@import url('https://fonts.googleapis.com/
css2?family=Dancing+Script&display=swap') ::
Importa do google fonts. 

font-family: 'Dancing Script','cursive'; ::
adicione a familia á cascata que deseja usar a fonte importada


    @font-face { /* Fonte TrueType */
        font-family: 'NomeFonte';
        src: url('fontes/lovelarry.otf')
        format('opentype');
    }
    
    @font-face { /* Fonte TrueType */
        font-family: 'NomeFonte';
        src: url('fontes/lovelarry.ttf')
        format('truetype');
    }

    h1{
        font-family: 'NomeFonte', sans-serif;
    }

    Talvez seja necessário colocar o font-weight e
    font-style como normal para padroniza-las.
<TIPOS DE FORMATO> ::

opentype (otf) - truetype (ttf) - embedded-opentype 
truetype-aat (Apple Advanced Typ) - (svg)
