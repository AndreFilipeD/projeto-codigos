<CSS3 : CAIXAS>
<DISPLAY>

display: block; :: transforma elemento em block-level
display: inline; :: transforma elemento em inline-level
display: none; :: esconde o elemento
display: inline; :: transforma elemento em flex

<TAMANHO>

<Tipos de manipulação>
100px :: manipulação em pixel.
100% :: preenche o espaço disponivel da caixa em 100%.
100vw :: tamanho = 100% da proporção horizontal da tela.
100vh :: tamanho = 100% da proporção vertical da tela.

width: 100px; :: largura.
height: 100px; :: altura.

min-width:50px e max-height:500px :: Define limites para a
distorção da caixa.

padding: 50px :: preenchimento interno.
margin: 50px :: espaçamento externo.


<BORDAS>

Tipos de bordas: solid, groove, dashed, dotted, double, ridge,
inset, outset.

border: 1px solid black; :: adiciona uma linha
de 1px solida e preta na caixa, tem outras propriedades...

outline: 1px solid black; :: adiciona uma linha externa
na margem externa da caixa, com propriedades da border.

border-radius: 20px; :: adiciona arredondamento na borda

box-shadow: 10px 10px 10px 10px black; :: 
1° posição x, 2° posição y, 3° blur, 4° espalhar, 5° cor.
inicie com inset para sombrear internamente.
