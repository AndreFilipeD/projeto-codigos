<CSS3 : GRID 1>
<HTML DE ESTUDO>

Considere para todos os exemplos Grid
a seguinte estrututa HTML ::

<div class="container">
    <div class="red"> </div>
    <div class="green"> </div>
    <div class="blue"> </div>
    <div class="yellow"> </div>
</div>
<CSS DE ESTUDO>

.container{
    width: 100px;height: 100px;
    border: 1px solid black;
    display: grid;
    grid-template-columns: 1fr 1fr;
    grid-template-rows: 1fr 1fr;
}
.red{ /* .green , .blue , .yellow */
    width: 40px;height: 40px;
    background-color: red;
} /* considere o mesmo, só que com outra cor...*/



    
 

    
 

    
 

    
 




<METODOS UTILIZADOS>

[Declarações]

display: grid; :: Define o container como grid container.
display: inline-grid; :: Também define grid porem inline.

[Grid templates]

grid-template-columns: 1fr 1fr; ::
Define quantidade de colunas, com suas respectivas larguras.

grid-template-rows: 1fr 1fr; ::
Define quantidade de linhas, com suas respectivas alturas.

<DUVIDAS>

Caso largura ou altura definida seja maior que o container::
• valor fixo em px :: por padrão as "celulas" utilizarão
suas respectivas dimensões, ignorando o tamanho do container

• 1fr :: Significa que a celula ocupará 1 fração do espaço total
disponivel no container, caso ainda haja espaço para ajuste.
