<REACT + CSS: INLINE STYLE> 
<SOBRE>

:: O CSS inline em React  é equivalente ao estilo do HTML inline,
declaramos o estilo por extenso dentro da tag, porem, faremos uso
de camelCase, e passamos um objeto dentro do JSX ou TSX, onde devemos
declarar os valores conforme o exemplo abaixo ::

<h1 style={{color: 'red', backgroundColor: 'blue';}}>texto de Exemplo</h1>

:: Vamos observar a tag h1 acima, os elementos são separados por virgula (,)
no lugar dos dois pontos tradicionais (;), já que está em JSX ou TSX;

também podemos ver em background-color, declarado como backgroundColor
que é o camelCase mencionado acima;

pra finalizar, repare que red e blue estão entre aspas, já que se
trata de uma atribuição de valor em um objeto.
