<REACT + CSS: OBJECT STYLE> 
<SOBRE>

:: Já que o JSX ou TSX recebem um objeto como parâmetro de estilo em css,
podemos criar um objeto com os estilos já definidos e simplesmente
atribui-los ao inline, conforme exemplo abaixo ::


const textoDestaque={
    color: 'red',
    backgroundColor: 'blue',
    fontSize: '2em'
}


<h1 style={{textoDestaque}}>texto de Exemplo</h1>

Apenas passando o nome do objeto com os parâmetros, já atribuiremos os
estilos definidos.
