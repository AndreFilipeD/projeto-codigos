:: Como já esperado, o ideal é sempre se organizar e fazer uma pasta
para realizar a crição de seus arquivos CSS, vamos utilizar o App.css
como exemplo, que se encontra em src > components, onde está o arquivo
principal App.js ou App.ts.

Basta importar o arquivo css conforme exemplo abaixo dentro do código
principal ::

import "./App.css"

Lembrando que se os estilos dos componentes estiverem nesse arquivo,
por se tratar do arquivo fonte do projeto, não será necessario
importar o mesmo arquivo em cada componente, apenas caso queira se
organizar de tal maneira em que cada componente especifico tenha sua
folha de estilo;

Considere o seguinte estilo dentro do documento css ::

.texto{
    font-size: 2em;
    color: green;
    background-color: red;
}

::

Agora para utilizar o estilo em React, devemos incluir sua ClassName ::

<h1 className="texto">Texto de exemplo</h1>

:: Lembrando que se selecionar a pasta public, encontrará o index.html,
no qual também pode importar o arquivo principal de estilo, ou
talvez customizar alguma coisa do próprio arquivo HTML como favicon..

pode aproveitar para customizar o estilo do body inline mesmo, já
que não há muitas maneiras de acessa-lo via JSX ou TSX
