<REACT : LIST EXPORT RENDER> 
<DESCRIÇÃO>
Podemos exportar listas juntamente com seus tipos em arquivos
separados, para que possamos renderiza-la posteriormente.

Vamos começar sempre organizando os arquivos dentro da pasta
src, criando dentro dela a pasta data, e dentro de data, o arquivo peopleList.ts

Acessando o arquivo, vamos criar a seguinte lista ::

export const peopleList = [
    {id: 1, name: 'Fulano'},
    {id: 2, name: 'Ciclano'},
    {id: 3, name: 'Beltrano}'
];

:: Considerando que temos um arquivo principal dentro de app,
faremos na primeira linha a seguinte importação ::

import { peopleList } from '@/data/peopleList';

:: Vamos criar mais uma pasta em src chamada types
e dentro dela um arquivo chamado Person.ts
contendo o seguinte conteudo ::

export type Person = {
    id: number;
    name: string;
}

:: Voltando no arquivo peopleList.ts vamos importar os tipos ::


export const peopleList: Person[] = [
    {id: 1, name: 'Fulano'},
    {id: 2, name: 'Ciclano'},
    {id: 3, name: 'Beltrano}'
];

:: Voltando para o código principal, vamos renderizar a lista usando
uma função do javascript chamada map() ::

import { peopleList } from '@/data/peopleList';

const Page = () => {
    return (
        <div>
            <h1 className="PeopleTitleList">Olá, Mundo!</h1>
            <h3>Algum outro texto</h3>

            <ul>
                {peopleList.map(person => <li> {person.name} </li>)}
            </ul>
        </div>
    )
}
