<PROPS PADRÃO>
Podemos também adicionar valor inicial nas props definidos na aba
Props 2, seguindo o exemplo abaixo ::


    export const Person = ({
        name='André',
        avatar='www.com.br/img',
        roles: Props) => {
    
        return{
            <>
    
                <h1>name</h1>
    
                <img
                    src={avatar}
                    alt={name}
                    className="w-40" //css
                />
    
                <ul>
                    <li>{roles[0]}</li>
                    <li>{roles[1]}</li>
                </ul>
    
            </>
        }
    }


:: Lembrando que name e avatar estão recebendo como padrão 
um valor, mas se no typescript, a declaração vazia não for opcional,
obrigatoriamente terá que incluir alguma coisa, para resolver isso
basta adicionar na declaração do tipo, no elemento name e avatar, como
opcional.
 ex ::


    type Props = {
        name?: string;
        avatar?: string;
        roles: string[];
    }
