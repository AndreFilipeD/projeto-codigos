<REACT : INSTALAÇÃO> 
<SOBRE>
O REACT é uma biblioteca JavaScript para criação de interfaces
de usuário, com foco no desempenho e na reutilização, e sua
arquitetura é baseada em componentes.
<INSTALANDO REACT NO WINDOWS>
Primeiramente precisamos instalar:

Node.js e npm (Node Package Manager) que são
compactados no mesmo pacote e podem ser encontrados aqui ::

https://nodejs.org/en/download/  Clique aqui para acessar -> 
Utilize esses 2 comandos no terminal para verificar se a
instalação acima funcionou ::
node -v e npm -v

Após, execute o seguinte comando ::
npm install -g create-react-app

Aguarde finalizar, pode demorar...

Após, execute para criar um projeto react no diretório atual ::
create-react-app meu-projeto

:: Substitua meu-projeto pelo nome que desejar para o diretório ::

Agora vá  para o diretório do projeto e inicie o servidor utilizando ::
cd meu-projeto :: e depois :: npm start :: para iniciar o servidor

também no prompt, lembre-se de encerrar o servidor ::
Pressione junto :: ctrl + c :: e após ser perguntado,
confirme se quer encerra-lo utilizando (S/N) 
<FINALIZAR PORTA 3000 OU OUTRA>

Se acontecer de você fechar tudo e se esquecer de finalizar a porta
do servidor, realize o seguinte processo ::

npx kill-port 3000 :: e confirme a "instalação" da
finalização do processo, PRONTO! estará devidamente morto.


Uma forma de remover um projeto react é ir pelo prompt até a pasta do
projeto e digitar o seguinte código ::
"rmdir /s node_modules"

A outra seria utilizando Yarn ::

Basta acessar a pasta do projeto pelo terminal e executar o comando
"yarn cache clean" ou "npm cache clean" para limpar o cache dos
pacotes instalados. Em seguida, execute o comando "yarn install"
ou "npm install" para reinstalar as dependências do projeto.
Isso deve remover a pasta node_modules antiga e criar uma nova.
