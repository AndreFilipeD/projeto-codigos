<REACT : INSTALAÇÃO> 
<SOBRE>
O REACT é uma biblioteca JavaScript para criação de interfaces
de usuário, com foco no desempenho e na reutilização, e sua
arquitetura é baseada em componentes.
<INSTALANDO REACT NO WINDOWS>
Primeiramente precisamos instalar:

Node.js e npm (Node Package Manager) que são
compactados no mesmo pacote e podem ser encontrados aqui ::

https://nodejs.org/en/download/  Clique aqui para acessar -> 
Utilize esses 2 comandos no terminal para verificar se a
instalação acima funcionou ::
node -v e npm -v

Após, execute o seguinte comando ::
npm install -g create-react-app

Aguarde finalizar, pode demorar...

Após, execute para criar um projeto react no diretório atual ::
create-react-app meu-projeto

:: Substitua meu-projeto pelo nome que desejar para o diretório ::

Agora vá  para o diretório do projeto e inicie o servidor utilizando ::
cd meu-projeto :: e depois :: npm start :: para iniciar o servidor

também no prompt ::
Pressione junto :: ctrl + c :: encerra o servidor.
