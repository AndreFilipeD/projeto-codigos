<REACT : COMPONENTE BASICO>
Duas maneiras de criar um mesmo componente chamado Page ::

function Page(){
    return(
          <div>
            <h1>Olá, Mundo!</h1>
            <h3>Testando 123</h3>
          </div>
      );
}

:: ou ::

const Page = () => {
    return(
          <div>
            <h1>Olá, Mundo!</h1>
            <h3>Testando 123</h3>
          </div>
      );
}

:: O componente só pode ter um elemento principal, no caso
estamos utilizando uma tag div, e ao final é exportado ::

export default Page; :: para ser acessado no arquivo principal
