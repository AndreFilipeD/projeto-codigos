<DESCRIÇÃO>
Podemos também remover uma tag inteira ao invez de apenas um
"pedaço" do conteudo contido dentro dela utilizando ::

<>
    <h3>Frase: "{phrase}" </h3>
    {author && <p>Autor: "{author ? author : "DESCONHECIDO"}" </p>}
</>

Caso author não exista, ele sequer vai criar o parágrafo
já que o mesmo está renderizado apenas em caso de condicional
verdadeira.
