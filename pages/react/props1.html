<REACT : PROPS 1> 
<O QUE É>
A vantagem do React é a capacidade de criar componentes já construidos para
a interface utilizando apenas um import e um nome correto?

Ex ::
<Person/>
Mas e se esse componente pudesse ser mais reaproveitado? e se
pudessemos passar parâmetros diretamente em sua criação? é pra
isso que serve os props.

Ex de criação reaproveitada de person ::
<Person 
    name="elon musk"
    avatar="https://wayimg.com/123"
    roles={['CEO Tesla', 'CEO SpaceX']}
/>

:: Uma vez tendo a estrutura logo abaixo de Person já definido, basta
sempre chamar o componente Person atribuindo os seus devidos
valores internos que ele vai ser criado com dados diferentes.

Ex da estrutura inicial de Person ::

type Props = {
    name: string;
    avatar: string;
    roles: string[];
}

export const Person = (props: Props) => {
    
    return{
        <>

            <h1>props.name</h1>

            <img 
                src={props.avatar}
                alt={props.name}
                className="w-40"
            />

            <ul>
                <li>{props.roles[0]}</li>
                <li>{props.roles[1]}</li>
            </ul>
        
        </>
    }

}

:: Perceba que o componente está acessando as propriedades de props
onde recebe os dados como se fosse um objeto. Dados esse que vem
da declaração de utilização do próprio componente
