<REACT : IMPORTAR COMPONENTES>
Componentes geralmente não são exportados com default,
apenas a própria Page é exportada dessa forma.

Considere agora um arquivo Geo.tsx, onde teremos os componentes
Square e Circle

Componentes podem ser exportados em sua própria inicialização ::

export const Square = () => {
    return(
        <div className="w-50 h-50 bg-orange-700 text-white">
            Quadradin
        </div>
    )
}

export const Circle = () => {
    return(
        <div className="w-50 h-50 bg-orange-700 text-white rounded-full">
            Circulin
        </div>
    )
}

E para importa-lo em Page, deve ser feito assim ::

import {Square} from '@/components/Geo'

Se houver mais de um, separe :: {Square, Circle}
<DEFAULT + COMPONENT>
Se por algum motivo você tiver que realizar um "default export" junto com
um "componente export" em um mesmo arquivo, conforme exemplo abaixo ::

const Square = () => {
    return( ...data )
}

export const Circle = () => {
    return( ...data )
}

export default Square;

:: A importação no arquivo Page fica assim ::

import Square, {Circle} from '@/components/Geo';
