<REACT : COMPONENTE BASICO>

Abra o arquivo page em src > app e inicie :: 
Duas maneiras de criar um mesmo componente chamado Page ::

function Page(){
    return(
          <div>
            <h1>Olá, Mundo!</h1>
            <h3>Testando 123</h3>
          </div>
      );
}

:: ou ::

const Page = () => {
    return(
          <div>
            <h1>Olá, Mundo!</h1>
            <h3>Testando 123</h3>
          </div>
      );
}

:: O componente só pode ter um elemento principal, no caso
estamos utilizando uma tag div, e ao final é exportado ::

export default Page; :: para ser acessado no arquivo principal
<COMPONENTE INTERNO>
Crie uma pasta component dentro de src, e depois
um arquivo Square.tsx e preencha-o ::

const Square = () = >{
    return(
        <div className="w-25 h-25 bg-orange-700 text-white">
            Texto qualquer
        </div>
    );
}
export default Square;

Agora no arquivo Page, importe o Square na primeira linha assim ::
import Quadrado from '@/components/Square';

:: @ referencia ao próprio diretório do arquivo.

dentro do componente Page, na div principal
adcione na ultima linha :: <Quadrado />
