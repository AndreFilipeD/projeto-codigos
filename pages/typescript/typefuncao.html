<TYPESCRIPT : TYPE FUNÇÃO> 
Podemos facilitar muito a vida na hora de criar
funções no typescript usando esse conceito ::

type MathFunction = (n1: number, n2: number) => number;

:: Vamos analisar essa estrutura.

Criamos um tipo chamado MathFunction, e dizemos a ele que
ele vai receber n1 tipo number e n2 tipo number, logo após,
apontamos => para o retorno da função que já deve ser também
um tipo number.

Dessa forma, podemos criar as funções abaixo de maneira mais
dinamica, confira ::

const somar: MathFunction = (n1, n2) => {
    return n1 + n2
}

const subtrair: MathFunction = (n1, n2) => {
    return n1 - n2
}

const multiplicar: MathFunction = (n1, n2) => {
    return n1 * n2
}

:: Muito melhor, não é? praticamente javascript puro dentro
do typescript!
