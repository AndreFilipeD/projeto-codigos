<TYPESCRIPT : TYPE LITERAL> 
E se por exemplo, o tipo de resultado especificamente 
fosse apenas 3 possibilidades? se o retorno 
só pudesse ser "small", "medium" ou "big"?
bem especifico não é? Para isso temos os tipos literais ::

function tamanhoElemento(
    altura: 'small' | 'medium' | 'big',
    largura: 'small | 'medium' | 'big'){
    ... processamento case 'small medium big' > retorno
}

Podemos também criar o tipo para economizar espaço ::
  
interface SizeElement {
    altura: 'small' | 'medium' | 'big';
    largura: 'small' | 'medium' | 'big';
}

ou 
    
type SizeElement = {
    altura: 'small' | 'medium' | 'big',
    largura: 'small' | 'medium' | 'big'
}


:: o resultado da função ficaria assim ::

function tamanhoElemento(props: SizeElement){
    ... processamento case 'small medium big' > retorno
}
