<TYPESCRIPT : INSTALAÇÃO> 
Para instalar o Typescript, precisamos do NodeJs instalado na maquina,

Para isso basta acessar o site: NOJE JS OFICIAL

E realizar a instalação da versão LTS sempre.

Após isso, no terminal digite npm --version para verificar
se a instalação do NODE foi efetuada com sucesso.
Agora estamos prontos para instalar o Typescript ::

npm install -g typescript ::

Utilize esse comando no terminal, para fazer o download
e instalação do typescript pelo Node em sua maquina.

tsc --version :: para testar se a instação foi efetuada com sucesso;
Arquivos typescript são criados com final ts ex ::

script.ts

Geralmente criamos uma pasta src para armazena-lo,
e depois o compilamos para javascript na pasta public.
Dentro da pasta com o arquivo Typescript já criado, devemos
abrir o terminal no diretório

tsc script.ts :: compila o typescript para javascript

alguma das informações incluidas no typescript vão sumir do
arquivo final javascript, pois ele já converteu no javascript.
tsc script.ts --outDir public ::

Considere o comando no prompt acima para definir o nome da pasta
onde será compilado o código javascript, onde public é o nome
da pasta.
Recomendado que sempre compile assim ::

tsc script.ts --noEmitOnError ::

Dessa forma ele não vai gerar o javascript ao ser encontrado um
erro na codificação do arquivo, que podes ser corrigido ainda no
Typescript
