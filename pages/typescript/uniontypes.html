<TYPESCRIPT : UNION TYPES>
<SIGNIFICADO>
E se uma variavel puder ser de 2 tipos? string e number por exemplo?
é para isso que serve Union Types.
<COMO UTILIZAR>

let idade: string | number = 90;

idade = window.document.getElementById('idade').innnerHTML;

::

Nesse exemplo teriamos problemas com os 2 tipos, pois ambos
são requisitados :: o numero na declaração, e a string ao
atribuir o innerHTML do elemento #idade dentro da variavel idade.

Mas utilizando a declaração com aquela barra em amarelo separando 2
ou mais tipos, podemos unir os tipos e permitir ambos os casos.
<TRATAMENTO DE ERRO>
Vamos considerar o exemplo abaixo ::

function mostrarIdade(nome: string, idade: string | number){

    if(typeof idade === 'string'){
        console.log(idade.toUpperCase());
    }else{
        console.log(idade);
    }

}:: Lembre-se, caso você defina uma propriedade com
dois tipos, e que tem uma função que demanda apenas um dos tipos,
sempre terá que separar o typeof para evitar que o typescript
aponte erros, conforme o exemplo dessa função.
