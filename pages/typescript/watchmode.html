<TYPESCRIPT : WATCHMODE>
Watchmode é uma maneira de, a cada vez que salvar o código 
Typescript, atualizará o código fonte javascript
<PROJEÇÃO>Execute no prompt de comando ::

tsc script.ts -w
ou
tsc script.ts --watchMode

Agora o WatchMode está o tempo todo verificando atualização
salva no typescript, e em seguida tentando compilar o código
para gerar dinamismo na produtividade do seu desenvolvimento;
