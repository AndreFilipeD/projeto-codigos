<TYPESCRIPT : FUNÇÕES> 
Essa função transforma o primeiro caractere de uma string
em maiusculo ::

function firstLetterUpperCase(name: string){
    let firstLetter = name.charAt[0].toUpperCase();
    return firstLetter+name.substring(1);
}

firstLetterUpperCase('andré') :: Resultado 'André' ::

Mas se utilizarmos um numero no lugar da string teremos um erro,
pois o processamento da função é feito sobre uma string. ex ::

firstLetterUpperCase(90) :: erro

Então, o : string inserido dentro da declaração da função acima
previne que esse tipo de problema ocorra.
